# アプリについて
製造現場で利用できる生成AIアプリを作りたい。
熟練しのノウハウ継承や、社内ドキュメントの参照、人間で難しい事象の分析、業務効率化を目指している。将来的にはSAPや設備データなど、上位下位とつながって、自律的に動けるようにしたい。

## 機能について
* 社内文書を参照出来るチャットボット
* 音声入力や設備データをもとに日報のフォーマット等に書き換えてくれる
* 異常検知、外観検査：マルチモーダルな入力データをもとに通常と異なる状態などを発見してくれる
* 動的な生産計画、組立員のシフトや部材の仕入れ状況など様々な情報から生産計画などを動的に計画してくれる

## 画面構成について
基本3列の構成（必ずしもではない）  
右側：根拠を示す
中央：チャット
左側：検索対象等ソースとなるファイル等
上部にメニューバーがあり、上記の4機能を切り替える

## 技術スタックについて

**バックエンド**: python
